# Continuous Heart String Art Animation

A creative Python Turtle Graphics project that generates a continuous heart-shaped string art animation using mathematical equations. The heart is formed through dynamically connected line segments, creating a smooth and visually appealing animation.

---

## Project Objective

This project demonstrates how mathematical functions and Turtle Graphics can be combined to create continuous string art. Instead of drawing a simple heart outline, the animation connects multiple points on a heart curve to produce an artistic heart pattern.

---

## Features

- ❤️ Continuous heart string art animation
- 🎨 Multiple color transitions
- 📐 Heart generated using mathematical equations
- ⚡ Smooth Turtle Graphics animation
- 🖥️ Beginner-friendly Python project
- 🔄 Infinite animation loop
